This is a simple pizza project folder entirely made using assembly arm7 code.
Each file is more advanced and more complicated than the previous.
Pizza1.s is a simple file asking people how many pizzas they would like.
Pizza2.s adds on to Pizza1.s showing the cost of pizzas ordered by customers.
Pizza3.s asks how many small, medium, large pizzas, customer would like. Prints total cost of Pizzas.
Pizza4.s adds a sum function that adds up total number of pizzas
Pizza5.s adds a total cost function for the number of pizzas ordered.
Pizza6.s adds branches that add numbers for each size of pizza
Pizza7.s adds a loop that will continue to add pizzas to the order until the customer states they are done ordering.
